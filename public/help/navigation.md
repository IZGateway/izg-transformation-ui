# Navigating the Console

## Application Shell

The Transformation Console has a two-panel layout:

- **Left navigation sidebar** — always visible; contains links to the three main feature
  areas.
- **Main content area** — displays the active page.
- **Application header** — a top bar that shows a welcome message with your name.

![Application shell](../images/navigation-shell.png)

## Header

The application header displays:
- Your **avatar** (a generic user icon).
- A **welcome message** — "Welcome to IZ Gateway Transformation Service, *Your Name*" —
  so you can confirm which account is signed in.

## Left Navigation

The sidebar contains three navigation items:

| Menu item | Icon | Page |
|---|---|---|
| [**Manage Pipelines**](pipelines/index.md) | Transform (rotated) | `/manage` |
| [**Solutions Creator**](solutions/index.md) | Rule | `/solutions` |
| [**Mapping**](mappings/index.md) | Map | `/mapping` |

Click any item to navigate to that section. The active section is highlighted in the
sidebar.

![Left navigation](../images/navigation-sidebar.png)

## Error Pages

### Application Error

If the console encounters an unhandled runtime error, a generic error page is displayed.
The error page includes a link to the IZ Gateway Help Desk — use it to report the issue
along with the steps that led to the error.

### Page Not Found (404)

If you navigate to a URL that does not exist, a 404 page is displayed. Click the
**Home** link or use the sidebar to return to a valid section.
